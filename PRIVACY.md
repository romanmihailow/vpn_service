Политика конфиденциальности MaxNet VPN

1. Общие положения

1.1. Настоящая политика описывает порядок обработки и защиты персональных данных пользователей сервиса MaxNet VPN.
1.2. Используя сервис, пользователь соглашается с данной политикой конфиденциальности.

2. Какие данные обрабатываются

2.1. Сервис обрабатывает только данные, необходимые для работы:

Telegram ID пользователя;

данные о подписке и сроке её действия;

технические данные VPN-подключения.

2.2. Сервис не запрашивает и не хранит паспортные данные, адрес проживания или платёжные реквизиты пользователей.

3. Цели обработки данных

Данные обрабатываются исключительно для:

предоставления VPN-доступа;

учёта подписок и платежей;

технической поддержки пользователей.

4. Передача данных третьим лицам

4.1. Данные не передаются третьим лицам, за исключением:

платёжных сервисов (в рамках обработки платежей);

случаев, предусмотренных законодательством.

5. Защита данных

Сервис принимает разумные технические и организационные меры для защиты данных от несанкционированного доступа.

6. Изменение политики

Политика конфиденциальности может обновляться. Актуальная версия всегда доступна на сайте сервиса.

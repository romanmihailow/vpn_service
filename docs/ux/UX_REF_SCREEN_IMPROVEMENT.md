# Улучшение экрана /ref (реферальная ссылка)

**Дата:** март 2025  
**Цель:** сделать экран /ref более понятным и конверсионным без изменения бизнес-логики и архитектуры реферальной программы.

---

## 1. Где добавлена кнопка "🤝 Пригласить друга"

- **Файл:** `app/tg_bot_runner.py`
- **Функция:** `cmd_ref(message: Message)` (обработчик команды `/ref`).

В конце функции, при отправке сообщения пользователю, добавлен `reply_markup`:

```python
text = "\n".join(lines)
await message.answer(
    text,
    disable_web_page_preview=True,
    reply_markup=REF_SHARE_KEYBOARD,
)
```

Клавиатура `REF_SHARE_KEYBOARD` уже существовала и содержит кнопку:

```python
REF_SHARE_KEYBOARD = InlineKeyboardMarkup(
    inline_keyboard=[
        [
            InlineKeyboardButton(
                text="🤝 Пригласить друга",
                callback_data="ref:open_from_ref",
            ),
        ],
    ]
)
```

Таким образом, под сообщением `/ref` теперь всегда есть явный CTA «🤝 Пригласить друга», который переиспользует существующий callback.

---

## 2. Какой callback используется

- Используется уже реализованный callback: `ref:open_from_ref`.
- Обработчик: `@router.callback_query(F.data == "ref:open_from_ref")` в `app/tg_bot_runner.py` (ранее использовался под другими реферальными экранами).
- Логика формирования текста для пересылки и реферальной ссылки **не менялась**.

---

## 3. Какой мотивационный текст добавлен

**Файл:** `app/tg_bot_runner.py`  
**Функция:** `cmd_ref`

Сразу после заголовка экрана `/ref` добавлен короткий мотивационный блок:

```python
# Заголовок
lines.append("👥 <b>Твоя реферальная ссылка</b>\n")
lines.append(
    "Приглашай друзей и получай бонусные дни VPN,\n"
    "когда они подключаются и оплачивают подписку.\n"
)
```

Этот текст кратко объясняет ценность реферальной программы до показа кода и ссылки.

---

## 4. Как скрыт админ-блок для обычных пользователей

**Файл:** `app/tg_bot_runner.py`  
**Функция:** `cmd_ref`

Блок с админской статистикой (`"🔧 <b>Админ:</b>"` и показатели `get_referral_admin_stats()`) уже был обёрнут в проверку `is_admin(message)`:

```python
if is_admin(message):
    lines.append("")
    lines.append("🔧 <b>Админ:</b>")
    try:
        stats = db.get_referral_admin_stats()
        ...
```

Поэтому для обычных пользователей этот блок **и ранее был скрыт**, а показывался только админу. В рамках текущего улучшения логика этого блока не изменялась; дополнительного кода для скрытия не потребовалось.
